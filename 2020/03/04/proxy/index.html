Proxy

Proxy是ES6新增的一种操作对象的API，用于定义基本操作的自定义行为（如属性查找，赋值，枚举，函数调用等）。

语法

let p = new Proxy(target, handler);

参数

  • target

    用Proxy包装的目标对象（可以是任何类型的对象，包括原生数组，函数，甚至另一个代理）。

  • handler

    一个对象，其属性是当执行一个操作时定义代理的行为的函数。

方法

Proxy有13种数据劫持的操作。

方法 描述
get 获取某个key值
set 设置某个key值
has 使用in操作符判断某个key是否存在
apply 函数调用，仅在代理对象为function时有效
ownKeys 获取目标对象所有的key
construct 函数通过实例话调用，仅在代理对象为function时有效
isExtensible 判断对象是否可扩展
deleteProperty 删除某一个property
defineProperty 定义一个新的property
getPrototypeOf 获取原型对象
setPrototypeOf 设置原型对象
preventExtensions 设置对象为不可扩展
getOwnPropertyDescriptor 获取一个自有属性(非原型连上的属性)的属性描述

  • get

    get方法是你在得到某个对象属性值时预处理的方法，接受两个常用参数

    • target：得到的目标值

    • key：对象的属性

  • set

    set方法用来拦截某个属性的赋值操作，可以接受四个参数

    • target: 目标值

    • key: 对象属性

    • value: 对象属性值

    • receiver: 改变前的原始值

Proxy相比Object.defineProperty的优势

  • 支持数组

  • 针对对象

    与Object.defineProperty()方法不同，Proxy针对的是完整的对象，而不是对象的某个属性，所以不需要对keys进行遍历。

  • 嵌套支持

    Proxy本质上是不支持嵌套的。可以通过在get方法里边递归调用Proxy来实现。
